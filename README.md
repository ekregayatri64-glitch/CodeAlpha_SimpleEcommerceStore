# CodeAlpha Simple E-Commerce Store

## Project Description

This is a Simple E-Commerce Store developed as part of the CodeAlpha Internship Program. The project allows users to browse products, view product details, add items to the cart, and place orders.

## Features

* Product Listing
* Product Details Page
* Shopping Cart
* Add to Cart Functionality
* Order Processing (Buy Now)
* Login Page
* Registration Page
* Responsive User Interface
* Backend API using Node.js and Express.js

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Local Storage

## Project Structure

* index.html – Home Page
* product.html – Product Listing
* details.html – Product Details
* cart.html – Shopping Cart
* login.html – User Login
* register.html – User Registration
* script.js – Frontend Logic
* style.css – Styling
* server.js – Backend Server

## How to Run

1. Install Node.js
2. Open terminal in the project folder
3. Run:
   npm install
4. Start the server:
   node server.js
5. Open index.html in your browser
